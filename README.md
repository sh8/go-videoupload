# go-videoupload
Easy video uploader for golang  
[![GoDoc](https://godoc.org/github.com/golang/gddo?status.svg)](https://godoc.org/github.com/sh8/go-videoupload)

# Install

```
$ go get github.com/sh8/go-videoupload
```
